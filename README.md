# theop-ScreenshotsCleaner
Cleans the screenshots folder of files older than 90 days old. 
This script only works on Windows.

# How to use
- Download the release zip folder from releases.
- Extract
- Run either the .bat file (Windows) or the python file
- The script can be automated using the built-in Windows Task Scheduler program

The deleted screenshots will appear in a folder in the recycle bin called "Old Screenshots"
